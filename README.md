#  Кафе "Солнечный зайчик"

![Flutter](https://img.shields.io/badge/Flutter-3.41.6-blue)
![Dart](https://img.shields.io/badge/Dart-3.11.4-blue)
![Firebase](https://img.shields.io/badge/Firebase-Auth-orange)

Мобильное веб-приложение для управления меню и заказами кафе. Администратор может добавлять и удалять блюда, пользователи — просматривать меню, добавлять в корзину и оформлять заказы.
### Требования
Flutter 3.41.6+ \
Dart 3.11.4+ \
Firebase проект (Auth + Firestore) 
### Запуск бэкенда
cd backend \
dart pub get \
dart run bin/server.dart

### Запуск фронтенда
cd frontend \
flutter pub get \
flutter run -d chrome 

## Функционал приложения
### Для всех пользователей:
Пользователь может создать аккаунт по email и паролю или войти в существующи, данные аутентификации хранятся в Firebase Auth. \
Главный экран показывает все блюда с фотографиями, названиями, описаниями и ценами, каждое блюдо отображается в отдельной карточке.\
В верхней части экрана есть поле поиска, которое позволяет найти блюдо по названию в реальном времени.\
Пользователь может отфильтровать блюда по категориям: напитки, обеды, салаты, ужины, десерты или посмотреть все.\
При нажатии на кнопку "в корзину" блюдо добавляется в корзину, в корзине можно увеличивать или уменьшать количество каждого блюда, а также удалять позиции. На иконке корзины в шапке приложения отображается общее количество выбранных товаров.\
При нажатии на кнопку "оформить" появляется форма, где нужно указать номер телефона и адрес доставки, после заполнения полей и нажатия "оплатить" заказ отправляется на сервер и сохраняется в базе данных.\
В отдельной вкладке пользователь может просмотреть все свои предыдущие заказы с деталями: состав заказа, общая сумма, статус, адрес доставки и дата оформления.\
В профиле отображается имя и email пользователя, также можно сохранить свой номер телефона и адрес доставки.\
Пользователь может переключаться между светлой и темной темой оформления приложения, кнопка переключения находится в профиле и на экране входа.\
В профиле есть кнопка выхода, которая завершает сеанс и возвращает на экран входа.\

### Для администратора:
Администратор имеет все возможности обычного пользователя плюс дополнительные функции управления меню.\
На главном экране меню у администратора появляется оранжевая плавающая кнопка с плюсом. При нажатии открывается форма, где нужно ввести название блюда, описание, цену и выбрать категорию, после добавления блюдо сразу появляется в общем меню.\
У каждой карточки блюда для администратора отображается красная иконка корзины, при нажатии появляется подтверждение, после чего блюдо удаляется из меню.\
Администратор может добавлять и удалять любые блюда в любое время, изменения сразу видны всем пользователям.
